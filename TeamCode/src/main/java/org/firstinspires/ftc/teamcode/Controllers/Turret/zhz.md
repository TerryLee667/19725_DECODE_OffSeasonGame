# 炮塔系统算法原理说明

## 1. Shooter（发射系统）算法原理

### 1.1 核心控制算法

Shooter类采用**PIDSVAController**进行发射电机的速度控制，这是一种增强版的PID控制器，包含以下组件：

- **PID控制**：比例(P)、积分(I)、微分(D)控制
- **前馈控制**：静态摩擦补偿(kS)、速度前馈(kV)、加速度前馈(kA)
- **电压控制**：通过VoltageOut类将计算的电压转换为电机功率

### 1.2 工作原理

1. **初始化**：
   - 配置左右发射电机，设置编码器模式和零功率行为
   - 初始化PIDSVAController，设置控制参数

2. **速度控制流程**：
   - 输入目标速度（targetSpeed）
   - 读取当前电机速度（通过编码器）
   - 使用PIDSVAController计算输出电压
   - 将电压转换为电机功率并应用
   - 检查速度误差是否在容差范围内

3. **关键参数**：
   - `kP`：比例系数，响应当前速度误差
   - `kI`：积分系数，消除稳态误差
   - `kD`：微分系数，抑制超调和振荡
   - `kS`：静态摩擦系数，补偿启动时的摩擦力
   - `kV`：速度前馈系数，根据目标速度预补偿
   - `kA`：加速度前馈系数，根据加速度预补偿

### 1.3 特点

- **双电机控制**：同时控制左右两个发射电机，保持速度一致
- **电压补偿**：考虑电池电压变化，确保输出功率稳定
- **实时参数调整**：通过Config注解支持FTC Dashboard实时调整参数
- **安全停止**：当目标速度为0时，直接停止电机

## 2. Turner（炮塔控制）算法原理

### 2.1 核心控制算法

Turner模块包含两个主要控制器：

- **TurretAimController**：根据目标位置计算并控制炮塔旋转角度
- **TurretDegreeController**：目前为空实现，计划用于直接控制炮塔角度

### 2.2 TurretAimController工作原理

1. **初始化**：
   - 初始化TurretAimingCalculator（瞄准角度计算器）
   - 配置炮塔电机，设置编码器模式和零功率行为
   - 重置炮塔角度基准

2. **角度计算流程**：
   - **步骤1**：接收机器人当前位置和方向角
   - **步骤2**：使用TurretAimingCalculator计算需要的旋转角度
   - **步骤3**：计算目标炮塔角度并归一化到[-180, 180]度
   - **步骤4**：控制电机旋转到目标角度
   - **步骤5**：从编码器读取实际角度并更新状态

3. **控制策略**：
   - 使用简单比例控制（P控制），根据角度误差调整电机功率
   - 当误差小于容差时停止电机
   - 支持高精度模式（使用RUN_TO_POSITION）

### 2.3 TurretAimingCalculator算法原理

这是炮塔瞄准的核心算法，负责计算炮台需要旋转的角度：

1. **坐标变换**：
   - 考虑炮台相对机器人中心的偏移（turretOffsetX, turretOffsetY）
   - 将机器人坐标系中的炮台偏移转换到世界坐标系

2. **目标位置计算**：
   - 计算目标相对于炮台的位置矢量（deltaX, deltaY）
   - 使用反正切函数（Math.atan2）计算需要的绝对瞄准角

3. **角度计算**：
   - 计算炮台当前的绝对朝向角（机器人方向 + 炮台相对机器人朝向）
   - 计算需要的旋转角度（目标角度 - 当前角度）
   - 归一化旋转角度到[-180, 180]度，保证最短旋转路径

4. **高度角度计算**：
   - 提供calculateTurretAxisAngle方法，根据目标高度计算仰角
   - 使用ToBoardDegree方法将目标距离转换为仰角（目前为占位实现）

### 2.4 关键参数

- `MOTOR_TICKS_PER_REV`：电机每转脉冲数
- `TICKS_PER_DEGREE`：每度对应的编码器脉冲数
- `MOTOR_MAX_POWER`：电机最大功率，避免转速过快
- `ANGLE_TOLERANCE`：瞄准容差，小于该值则停止电机
- `turretOffsetX/Y`：炮台相对机器人中心的偏移量

## 3. 系统集成

### 3.1 炮塔系统整体架构

1. **Turret类**：作为炮塔系统的主控制器，整合：
   - Shooter（发射系统）
   - TurretDegreeController（角度控制）
   - AprilTag视觉系统（目标检测）
   - 仰角电机控制
   - 发射机构伺服控制

2. **工作流程**：
   - 通过AprilTag检测目标
   - 计算目标角度并控制炮塔旋转
   - 计算发射参数（使用RK4求解器）
   - 控制发射电机达到目标速度
   - 触发发射机构发射小球

### 3.2 待实现功能

- **TurretDegreeController**：需要实现旋转到目标角度和获取当前角度的逻辑
- **仰角控制**：需要完善基于编码器的仰角精确控制
- **自动瞄准**：需要优化AprilTag自动瞄准逻辑，提高跟踪精度
- **弹道计算**：需要完善RK4求解器的使用，提高发射精度

## 4. 技术亮点

1. **先进的控制算法**：
   - PIDSVAController结合了PID控制和前馈控制，提高了速度控制精度
   - 坐标变换和角度归一化确保了炮塔旋转的准确性和高效性

2. **模块化设计**：
   - 清晰的职责分离，便于维护和扩展
   - 支持参数实时调整，方便现场调试

3. **多传感器融合**：
   - 结合编码器、IMU和视觉传感器，提高定位和瞄准精度
   - 考虑了炮台偏移等硬件特性，提高了算法的实用性

4. **安全设计**：
   - 速度容差判断，确保发射电机稳定运行
   - 电机零功率行为设置，提高系统安全性
